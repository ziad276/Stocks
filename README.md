# Stocks

A stocks management web application built with **ASP.NET Core MVC**, developed as a 
hands-on project alongside an ASP.NET Core course. The project is continuously improved 
as new concepts are covered.

## Tech Stack
- **Backend:** C#, ASP.NET Core MVC
- **Architecture:** Service layer pattern with interface contracts + Dependency Injection
- **Frontend:** HTML, CSS, JavaScript

## Project Structure
- `ServiceContracts/` — Service interface definitions
- `Services/` — Concrete service implementations
- `Stocks/` — Controllers, Views, Models

## Getting Started
1. Clone the repo
2. Open `Stocks.sln` in Visual Studio
3. Run with `Ctrl+F5`
