# Stocks

A full-stack web application built with **ASP.NET Core MVC** for tracking and managing stock data.

## Tech Stack
- **Backend:** C#, ASP.NET Core MVC
- **Architecture:** Service layer pattern with interface-based contracts (DI / IoC)
- **Frontend:** HTML, CSS, JavaScript

## Project Structure
- `ServiceContracts/` — Interfaces defining the service layer contracts
- `Services/` — Concrete service implementations
- `Stocks/` — MVC application (Controllers, Views, Models)

## Getting Started
1. Clone the repo
2. Open `Stocks.sln` in Visual Studio
3. Run the project (`Ctrl+F5`)
